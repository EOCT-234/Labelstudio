Introducción:

En este trabajo se usó https://labelstud.io/ para realizar la anotación de imágenes usando la segmentación por polígonos, dentro del cual se usaron las clases o etiquetas hoja, escala y peciolo para segmentar partes de cada imagen. En este trabajo se explica todo el trabajo paso a paso.

Objetivo:

  • Realizar la segmentación por polígonos de 25 imágenes de hojas lúcuma.

Material:

  • Laptop

  • Lapicero

  • Instalador de anaconda

  • Cógido de label studio para instalación

  • 25 imágenes de hojas de lúcuma

Procedimiento:

Paso 1.

Descargar el instalador de Anaconda en https://www.anaconda.com/download/success (Figura 1), e instalarlo (ejecutar con anticlic y “como administrador”, dejar todo por defecto)

Figura 1: Descarga de Anaconda Distribution

Paso 2.

Buscar “Anaconda Prompt” y abrir la intefaz (Figura 2).

Figura 2: Anaconda Prompt (clic en “abrir”)

Paso 3.

  • Buscar https://labelstud.io/ y copiar el código “pip install -U label-studio” (Figura 3) y dar Enter.

Figura 3: Ejecución de código: “pip install -U label-studio”

  • Cuando finalice ejecutar “label-studio” (Figura 4) y dar Enter.

Figura 4: Ejecución de código: “label-studio”

Paso 4.

Ver Figura 5.

  • Dar un nombre: PAD_LUCUMA

  • Importación de datos: https://drive.google.com/drive/folders/1pv3_1awIPfNOG9ZR2jCWH8xkxYaGVREK?usp=drive_link

  • Configuración de etiquetado -> Visión por computadora -> Segmentación semántica por polígonos

Figura 5: Configuración dentro de labelstudio

  • Segmentación semántica por polígonos, dentro de este agregamos las etiquetas que tiene la imagen: hoja, escala y peciolo.

  • Finalmente clic en “Save” o “Ahorrar”.

Paso 5.

Clic en la imagen y se abrió la ventana para seleccionar los polígonos por etiquetas (Figura 6).

Figura 6: Slección de polígono con su etiqueta.

Paso 5.

Clic en una etiqueta y luego cerrar el polígono, finalizando clic en “Update” para que se grabe como tal (Figura 7)

Figura 7: Grabación de polígonos.

Y repetir el procedimiento con todas las imágenes.

Resultados:

Para ver los resultados ejecute Anconda Prompt, ejecute la linea “label-studio”
